# Privacy Policy

This application respects and protects the privacy of all users.

## Information Collection
This app does NOT collect, store, or transmit any personal data.
All data is stored locally on the user's device.

## Data Usage
No data is uploaded to any server.
No third-party analytics or tracking tools are used.

## Permissions
Camera access is used only for QR code scanning.
Photo library access is used only for importing QR codes.
